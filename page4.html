When you have gotten a character card, the actions or the skills(except KKGs) which are on your card can be found here. Skills on this page can also be bought using xPs. The xPs of skills are stated at the end of the move.

TYPES OF SKILLS

There are 6 different type of skills the skill type is stated in a bracket after the name of the skill. The 6 types are as follows;

(trap)Normal Skill: these are skills that activate when a certain condition is met either in opponents or users turn although user can choose to activate it or not.

(atk) Attacking Skill: these type of skills deal damage to opponents

(time) Time Skill: these are skills that take more than one turn to fufll their purpose

(def) Defensive Skill: these are skills that prevent you from losing hp such as sheild ,barriers etc

(neg) Neglecting Skill: these skills stop effect of other skills base on condition

(gain) Gain Skill: these types of skill increase your stats, hp or cp or even allow you play more than once.


Note: A skill slot can only hold one skill, but when the slot holds a mode, it can contain 3 modes.If a turn is skipped you lose 2hp but gain 2cp.You can only use a particular skill once, unless it is stated in the description of the skill that it can be used more than once. But some skills take turns to recharge for use again thuo.A skill can be wasted, which means, if the move is used in a wrong situation, the move is counted but will have no effect in the battle.To use a skill against a player, will be explained later on. A move can be sold for half the XPs it is worth. they are also skills that require certain weapons to be able to use them

Note if your hands are cut off you wont be able to use skills that requires hand, the admin will be the judge of such skills. Physical attacks are attacks like punch ,kick, non enchanted weapon skills and so on that don’t use cp.
